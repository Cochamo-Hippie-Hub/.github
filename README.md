## 🌿 What is this?

**Cochamo Hippie Hub** is an open-source platform for territorial management in conservation areas — born in the **Cochamó Valley**, Patagonia, Chile.

Often called *"the Yosemite of South America,"* this valley is at the center of the **Conserva Pucheguín** campaign: **USD $78 million** raised, **133,000 hectares** protected forever.

We build tools *from inside* the territory, *with* the people who live there.

> **`Naveguemos la vanguardia.`** — *Let's navigate the frontier.*

## 🗓️ Conservation timeline

| Year | Milestone |
|:-----|:----------|
| **2013** | 🟢 Puelo Patagonia founded — defense of Río Puelo from Mediterráneo hydroelectric project |
| **Jan 2023** | 🟢 Valle Cochamó declared **Nature Sanctuary** — 11,400 ha, unanimous vote |
| **Jan 2024** | 🟢 Contraloría ratifies Sanctuary decree — official legal protection in force |
| **Apr 2024** | 🟡 **Conserva Pucheguín campaign launched** — coalition: CPP + TNC + Freyja + Patagonia + Wyss |
| **Aug 2025** | 🟡 **Fundación Conserva Pucheguín constituted** — independent Chilean nonprofit |
| **Oct 2025** | 🟢 **Río Puelo declared Water Flow Reserve** — first river protected under 2022 Water Code Reform |
| **Dec 2025** | 🔴 **Historic purchase: $78M raised, 133,000 ha secured** — Fundo Pucheguín is no longer for sale |
| **Feb 2026** | 🔵 **"Cochamó por Siempre"** film released — Patagonia Films, narrated by Sean Villanueva |

<table>
<tr>
<td align="center" width="33%">

### 📡 Field capture
<sub>Ecological sensors · Satellite monitoring<br/>(Sentinel / Landsat / SAR)<br/>Visitor registration · LoRa / Starlink</sub>

</td>

<sub>Species ID via AI · Fire detection<br/>Vegetation change analysis<br/>Visitor flow modeling · Carbon quantification</sub>

</td>
